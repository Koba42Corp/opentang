March 30, 2026. 14 hours that shipped a product.

💬 The Spark 09:34 AM

A conversation that started a product

Admiral Dracattus sent the first message: build me an installer for my self-hosted stack. No brief. No spec. Just a vision and a deadline of "let us build this."

📋 Planning 10:40 AM

PRD, Design System & Build Checklist

Before writing a single line of code: a full Product Requirements Document, a complete design system (dark theme + orange), and a 100-task build checklist. Shipped in 66 minutes.

🚀 M1 — Bones 11:29 AM

Tauri v2 scaffold, all 9 wizard steps

First commit: a complete Tauri v2 + React + TypeScript scaffold with the full wizard shell, design system applied, fonts loaded, and every step navigable. Zero to buildable in one shot.

e011fa2
🔍 M2 — Heartbeat 12:56 PM

Real system detection goes live

The app now reads your actual machine: Docker version, RAM, disk space, WSL2 status. All via sysinfo Rust crate. The wizard blocks on failures and guides you through fixes.

0f9849c
🧙 M3 — The Wizard 13:04 PM

Full interactive UI, all 9 steps

All 9 wizard steps fully built: Edition picker, LLM config (Ollama + cloud), package selection with live resource estimator, network mode, credential generator, review screen, and install progress.

1a8e9a5
🩸 M4 — First Blood 13:21 PM

Real Docker Compose engine

The install button now actually installs. A Rust engine generates production-quality docker-compose.yml files, runs docker compose up -d, and streams real container logs back to the UI in real time.

fa6f97a
📦 M5 — Full Stack 13:39 PM

All 14 services. SSL. Health checks.

Coolify, OpenClaw, Gitea, Grafana, Prometheus, n8n, Uptime Kuma, Vaultwarden, Nextcloud, SearXNG — all with production Compose templates, Traefik SSL, and post-install health polling.

30162bb
🏪 M6 — Registry 13:40 PM

App Store. Dashboard. State persistence.

Post-install management: a full App Store with category browsing, install/remove/update per package, a live Dashboard with container health, and state persistence so the app remembers your stack.

49e5b65
🚢 M7 — Ship It 13:53 PM

GitHub Actions release pipeline

4-platform CI matrix: Linux, macOS Intel, macOS Apple Silicon, Windows. GitHub Actions builds signed installers on every tag. README polished. Auto-update wired.

d73fa45
🐳 Feature 14:22 PM

Docker auto-install added

Users without Docker get a single "Install Docker" button. On Linux it runs the official get.docker.com script. On macOS it opens OrbStack. No prerequisites required.

4ae47b2
🍊 Brand 14:57 PM

Logo integrated throughout the app

The official OpenTang citrus logo — wordmark for hero screens, mark-only for nav and icons — deployed across Welcome screen, sidebar, management shell, and all platform icons.

04654dc
🌐 Live 14:58 PM

opentang.koba42.com goes live

Landing page deployed to GitHub Pages via Cloudflare. Dark theme, orange branding, full feature sections, EULA, TOS, and Sage Wallet as the first Chia ecosystem integration.

🐛 Beta Tested 16:25 PM

4 bugs found. 4 bugs fixed. Same hour.

A community beta tester (with Jarvis AI) found: YAML indentation corruption, $ in passwords breaking .env, non-existent Docker images, and a port conflict. All four diagnosed and patched within 60 minutes.

⛓️ Ecosystem 19:11 PM

Sage Wallet joins the App Store

Sage Wallet — the premier Chia blockchain wallet — added to the OpenTang registry with a dedicated Crypto category. First third-party desktop app integration.

🔧 Fix 21:32 PM

macOS Docker PATH bug resolved

Discovered: macOS GUI apps inherit a minimal PATH that excludes /usr/local/bin. OpenTang now searches 5 common Docker install locations. 96GB RAM now reports correctly too.

🛡️ v0.1.1 02:32 AM (+1)

Pre-install detection ships

v0.1.1: OpenTang now scans running Docker containers before install. Existing services are detected, their ports imported, and they are skipped from reinstall — with a Force Reinstall override for power users.
